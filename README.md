# Quickstart

```
git clone ...
cd expo-activity-feed-demo
cp env.example.js env.js
```
## Create Stream account

https://getstream.io/

## Modify credentials

Edit env.js:

```
const STREAM_API_KEY = ''
const STREAM_API_SECRET = ''
const STREAM_APP_ID = ''
const STREAM_API_TOKEN = ''
```

## Install and start the app
```
yarn
yarn start
```
